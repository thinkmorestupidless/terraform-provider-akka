# akka_hostname

Registers a custom hostname for a project on the Akka platform. After registration, point your
DNS `CNAME` record to the value provided by the Akka platform, then attach a TLS secret to
enable HTTPS.

## Example Usage

```terraform
resource "akka_hostname" "api_domain" {
  hostname   = "api.example.com"
  project    = akka_project.example.name
  tls_secret = akka_secret.tls.name
}

output "hostname_status" {
  value = akka_hostname.api_domain.status
}
```
